# Product Vision: SDD Dry Days

## Overview
SDD Dry Days is a personal wellness tracker focused on helping individuals monitor and celebrate their alcohol-free days ("dry days"). The application provides motivation through visualization, goal setting, and progress tracking.

## Problem Statement
Many people want to reduce their alcohol consumption or track periods of sobriety, but lack simple, motivating tools to:
- Record and visualize their dry days
- Set personal goals and track progress
- See long-term patterns and trends
- Stay motivated through visual feedback

## Target Users

### Primary User (Current)
- **Individual user**: Person tracking their own dry days for personal wellness
- **Characteristics**: Self-motivated, wants simple daily tracking without complexity
- **Usage pattern**: Quick daily check-ins, periodic progress reviews

### Future Users
- **Friends/Family**: Trusted individuals who also want to track dry days
- **Support network**: People who want to share progress and encourage each other
- **Characteristics**: May want privacy controls, social encouragement features

## Core Features

### MVP Features (Current Focus)
1. **Add Dry Days**
   - Quick entry for today (default)
   - Ability to enter specific dates or date ranges
   - Simple, frictionless input

2. **Calendar Visualization**
   - Visual calendar view showing dry days
   - Clear, colorful representation of progress
   - Easy to see patterns at a glance

3. **Goal Setting & Tracking**
   - Set streak goals (e.g., "10 dry days in a row")
   - Track progress toward active goals
   - Visual progress indicators

4. **Progress Statistics**
   - Summary stats: dry days in last 30/60/90 days
   - Percentage calculations
   - Trend visualization

### Future Features
- **Theming**: Customizable color themes for console output
- **Sharing**: Share progress with friends/family
- **Social features**: Encourage others, celebrate milestones together
- **Export/Reports**: Generate progress reports

## Success Metrics

### Primary Success Indicators
- **Regular usage**: User logs dry days consistently
- **Motivation boost**: User reports feeling more motivated to maintain dry days
- **Goal achievement**: User successfully reaches set goals
- **Habit formation**: App becomes part of daily/weekly routine

### Future Success Indicators
- **User retention**: Continued usage over months
- **Social adoption**: Friends/family join and stay active
- **Goal progression**: Users set and achieve increasingly ambitious goals

## Product Principles

1. **Simplicity First**: Make adding a dry day effortless
2. **Visual Motivation**: Use colors and patterns to celebrate progress
3. **Privacy Focused**: Personal data stays personal (no forced sharing)
4. **Non-Judgmental**: Focus on positive progress, not failures
5. **Encouraging**: Celebrate every dry day, no matter how many

## User Journey

### Daily Flow
1. User opens the app
2. Marks today as a dry day (one command/action)
3. Sees immediate visual feedback and current streak
4. Closes app (total time: <30 seconds)

### Weekly Review
1. User opens calendar view
2. Reviews week's pattern visually
3. Checks progress toward goals
4. Adjusts goals if needed

### Monthly Reflection
1. User views 30/60/90 day statistics
2. Celebrates milestones
3. Sets new goals for next period
